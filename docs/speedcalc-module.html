Module speedcalc

source code

Scientific Programming for Earth and Ecological Discovery (SPEED) Calculation module created for the class.

This contains the core calculations and code for all SPEED Modules.

Note: As there are far more advanced [further abstraction, more dynamic, more feature-rich] libraries out there for unit conversion, this was just done as an exercise to familiarize myself with certain constructs that I am used to from other languages (e.g. Java/C#), help meet specific needs for a class and get general practice at making something from scratch.

GitHub repository: https://github.com/maplion/SPEED
